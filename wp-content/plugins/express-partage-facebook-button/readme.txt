== Description ==

create and customize a share button on FaceBook, this pulgin also save statistics of Facebook sharing

== Installation ==

Follow the steps below to install the plugin.

1. Upload the Facebook-share directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Express Partage FB" option to configure the button
